# Dice Prototype Game

Это приложение реализует простую игру в кубики с возможностью создавать клоны исходного кубика и применять модификаторы к ним.

В проекте реализованы два подхода к копированию кубика:
1. **DeepCopy** — создавалась полная копия объекта Dice через отдельный метод (DeepCopy), использовалась в GameManager для генерации клонов.
2. **Prototype (Clone)** — в новом варианте Dice реализует метод `Clone()`, который создаёт новый объект с копией всех граней. GameManager использует именно этот метод для порождения новых клонов.

Разница между подходами в данном проекте незначительная, однако паттерн **Prototype** официально внедрён и демонстрирует, как можно клонировать объекты без зависимости от конкретного конструктора и внутренней структуры класса.
